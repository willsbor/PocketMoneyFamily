# PocketMoney Family Static Site

This repository hosts the public static pages for App Store submission and support.

## Pages

- `/` - landing page
- `/privacy/` - privacy policy
- `/privacy-choices/` - privacy choices
- `/delete-account/` - account deletion
- `/support/` - support

## Notes

- The pages are built with plain HTML and a shared CSS file.
- Replace placeholder contact details before publishing.
- This repository is ready for GitHub Pages from the repository root.
